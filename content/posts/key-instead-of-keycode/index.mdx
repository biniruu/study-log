`keyup`이나 `keydown` 이벤트로 키보드 입력을 받을 경우, 이전에는 `event.keyCode`를 사용했었다.

<br />

```javascript
document.addEventListener("keyup", (evt) => {
  evt.preventDefault();
  // esc 키를 입력할 경우
  if (evt.keyCode === 27) {
    // 동작 구현
  }
});
```

<br />

이제 `event.keyCode`를 사용하는 방법은 지양해야 한다. `keyCode`가 deprecate 되었기 때문이다.

앞으로는 `event.key`를 사용해야 한다.

<br />

```javascript
document.addEventListener("keyup", (evt) => {
  evt.preventDefault();
  // esc 키를 입력할 경우
  if (evt.key === "Escape" || evt.key === "Esc") {
    // 동작 구현
  }
});
```

<br />

아직 `key`를 지원하지 않는 브라우저가 있다고 하는데, 지금은 ie를 제외하면 모두 지원하는 듯하다.

[KeyboardEvent.key | Can I use... Support tables for HTML5, CSS3, etc](https://caniuse.com/keyboardevent-key)

이전에는 아래 예시처럼 `key`와 `keyCode`를 모두 작성하라는 글이 많았는데, 이제는 `key`만 써도 될 듯하다.

<br />

```javascript
document.addEventListener("keyup", (evt) => {
  // esc 키를 입력할 경우
  if (evt.key === "Escape" || evt.key === "Esc" || evt.keyCode === 27) {
    // 동작 구현
  }
});
```
